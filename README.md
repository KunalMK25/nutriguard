# 🌿 NutriGuard — AI Malnutrition Risk Predictor

> Predicting malnutrition risk BEFORE it occurs — for children under 5 
> and pregnant women across India.

## Problem
India accounts for 35% of the world's stunted children. All existing 
tools detect malnutrition after irreversible damage has occurred. 
No tool predicts risk before it happens.

## Solution
NutriGuard is an AI-powered web app that combines:
- Parental genetic risk proxies (questionnaire-based)
- Regional cuisine & micronutrient mapping
- Maternal health indicators
- Socioeconomic factors

...to generate a 0-100 risk score with plain-language explanations, 
deployable by Anganwadi workers on any Android phone — offline.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS (PWA, works offline)
- Backend: FastAPI (Python)
- ML Model: XGBoost + SHAP Explainability
- Database: Supabase (PostgreSQL)
- Notifications: WhatsApp API (Twilio)
- Dataset: NFHS-5 (National Family Health Survey)
